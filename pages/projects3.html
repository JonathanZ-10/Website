Confundo

The Confundo project emulates a TCP connection over UDP with congestion
control, allowing a file to be sent from client to server through a reliable
data transfer protocol.

This project was rather complicated and required a vast knowledge of Python
BSD sockets. The goal was to make it possible to send a file without losses
even when sent over a lossy link with delay. This required setting up
the proper SYN ACK and FIN flags and ensuring that all data was stored
when transferred thorugh a UDP payload of 424 bytes.

Language: Python
IDE: PyCharm
Project Length: ~2 weeks
Team Size: 3 members
