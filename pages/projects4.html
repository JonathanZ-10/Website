Riddikulus Router

The Riddikulus project establishes a simple router with a static routing table
which handles Ethernet frames, ARP packets, IPv4 packets, and ICMP packets.
It is capable of forwarding as well as creating new frames, the client is also
able to ping all servers propely with the correct TTLs.

This project used Python BSD sockets and required a combination of mininet
and vagrant in order to create a virtual router to test the code.
This has been the most difficult project I have had to tackle and was exceedingly
difficult to get all of the code working in harmony.

Language: Python
IDE: PyCharm
Project Length: ~3 weeks
Team Size: 3 members
For more information please visit github or use a computer

Note: This project is private, if you'd like to see the code please contact me with your github username or github email
